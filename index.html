# What is KetamineUI?

KetamineUI is a feature-rich interface library designed for Roblox exploit script hubs. It provides a complete set of UI components — windows, tabs, buttons, toggles, sliders, dropdowns, and more — all wrapped in a sleek, customizable dark theme.

Unlike other libraries, KetamineUI is built from the ground up with:

  • 🔒 Anti-detection measures
  • 🎨 Deep theme customization
  • ⚡ Minimal performance impact
  • 📱 Responsive design that works on all screen sizes
  • 💾 Built-in configuration saving
  • 🔑 Integrated key system support

# Quick Example

Get up and running in seconds. Here's a minimal example that creates a window with a single button:

lua 
local KetamineUI = loadstring(game:HttpGet('http://localhost:3000/ketamineUI.lua'))()

local Window = KetamineUI:CreateWindow({
    Name = "💉 My Script Hub",
    LoadingTitle = "Loading...",
    Theme = "Purple",
    ConfigurationSaving = {
        Enabled = true,
        FolderName = "MyHub",
        FileName = "Config"
    }
})

local Tab = Window:CreateTab("Main", "home")

Tab:CreateButton({
    Name = "Click Me",
    Callback = function()
        print("Button clicked!")
    end
})

Windows

Tabs
