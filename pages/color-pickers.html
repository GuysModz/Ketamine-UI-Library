Color Pickers

Select colors dynamically for ESP and overlays.

Creating Color Pickers

lua
local ColorPicker = Tab:CreateColorPicker({
    Name = "Chams Color",
    Color = Color3.fromRGB(140, 60, 220),
    Callback = function(Value)
        print("Selected Color:", Value)
    end
})
